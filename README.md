# Rating App Solution

## Prerequisites
- Node.js installed
- PostgreSQL installed and running

## Database Setup
1. Create a database named `rating_app`.
2. Run the SQL commands provided in `database_schema.sql` (Create this file and paste your SQL query there!).

## How to Run
**1. Backend:**
cd rating-app-backend
npm install
npm run dev

**2. Frontend:**
cd rating-app-frontend
npm install
npm run dev
